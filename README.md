# Algorithm Study Records

개인적인 알고리즘 문제 풀이 및 학습 내용을 기록하는 저장소입니다. **백준허브(BaekjunHub)** 익스텐션을 사용하여 플랫폼별 풀이 내역을 자동으로 관리하고 있습니다.

## 📁 Folder Structure Guide

레포지토리의 주요 폴더 구성과 내용은 다음과 같습니다.

* **`백준/` (BOJ)**: 백준 온라인 저지에서 풀이한 문제들이 난이도별로 자동 저장됩니다.
* **`프로그래머스/` (Programmers)**: 프로그래머스에서 풀이한 문제들이 레벨별로 자동 저장됩니다.
* **`CPS/`**: 인프런 인강 **"it 취업을 위한 알고리즘 문제풀이 입문(C/C++) : 코딩테스트 대비"**의 연습 문제 풀이입니다.
* **`ProblemSolvingPractice/`**: 서강대학교 학교 수업 **"CSE4152: 문제해결프로그래밍실습"**에서 출제된 과제 및 문제 풀이입니다.
* **`BOJ/`**: (이전 기록) 백준 문제 풀이 중 직접 수동으로 관리했던 코드들이 포함되어 있습니다.
